## 📌 Overview

SmartMandi-AI is a Machine Learning based decision-support system designed to help farmers, traders, and agricultural stakeholders make smarter selling decisions.

The system predicts crop prices across multiple cities in Pakistan and recommends the best city to sell crops for maximum expected profit.

---

## 🎯 Problem Statement

Farmers often face:

- Uncertain crop prices  
- Lack of market awareness  
- Difficulty comparing city markets  
- Poor selling decisions  
- Profit loss due to limited information  

SmartMandi-AI solves this using predictive analytics and AI.

## 🧠 Machine Learning Model Used

### Random Forest Regressor

Why chosen:

- High prediction accuracy  
- Handles non-linear relationships  
- Works well on tabular data  
- Reliable for structured datasets
